# PR preview environments

`trip-canvas` preview environments are generated by an Argo CD `ApplicationSet` that watches open GitHub pull requests.

## CI behavior

GitHub Actions builds preview images only for pull requests whose source branch lives in the same repository.

Preview image tag format:

```text
ghcr.io/seonwook61/travlecanvas:pr-<number>-<head_sha>
```

## Shared preview secrets

Preview apps reuse one shared namespace and one shared secret set:

- Namespace: `trip-canvas-preview`
- Secrets:
  - `ghcr-credentials`
  - `trip-canvas-secrets`

Create them once:

```powershell
$env:GHCR_USERNAME='seonwook61'
$env:GHCR_TOKEN='<ghcr-read-token>'
$env:SUPABASE_SERVICE_ROLE_KEY='<preview-or-staging-supabase-secret>'
$env:GOOGLE_MAPS_SERVER_API_KEY='<preview-or-staging-google-server-key>'

kubectl create namespace trip-canvas-preview --dry-run=client -o yaml | kubectl apply -f -

kubectl create secret docker-registry ghcr-credentials `
  --namespace trip-canvas-preview `
  --docker-server ghcr.io `
  --docker-username $env:GHCR_USERNAME `
  --docker-password $env:GHCR_TOKEN `
  --dry-run=client -o yaml | kubectl apply -f -

kubectl create secret generic trip-canvas-secrets `
  --namespace trip-canvas-preview `
  --from-literal=SUPABASE_SERVICE_ROLE_KEY=$env:SUPABASE_SERVICE_ROLE_KEY `
  --from-literal=GOOGLE_MAPS_SERVER_API_KEY=$env:GOOGLE_MAPS_SERVER_API_KEY `
  --dry-run=client -o yaml | kubectl apply -f -
```

## Argo CD GitHub token

The pull request generator needs a GitHub token secret in the `argocd` namespace:

```powershell
$env:GITHUB_PR_GENERATOR_TOKEN='<github-token-with-repo-read>'
powershell -ExecutionPolicy Bypass -File .\scripts\deploy\Apply-PreviewTokenSecret.ps1
```

## Apply the ApplicationSet

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\deploy\Apply-PreviewApplicationSet.ps1
```

## Verify generated apps

```powershell
kubectl get applications -n argocd
kubectl get all -n trip-canvas-preview
```

## Local access

Use port-forward to inspect a preview app:

```powershell
kubectl port-forward svc/trip-canvas-pr-123 -n trip-canvas-preview 3000:80
```
